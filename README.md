# 🎮 Rock Paper Scissors Game

> A fun and interactive **Rock Paper Scissors** game built using **Java**, where you play against the computer 🤖.

---

## ✨ Features

* 🔁 Multiple rounds (loop added)
* 🤖 Random computer moves
* 🧠 Smart win/lose logic
* 🧮 Score tracking system
* 🚪 Exit option anytime
* 📊 Score tracking system
* ❌ Exit option (0)
* ⚠️ Invalid input handling
* 🎯 User-friendly output (Rock/Paper/Scissors instead of numbers)

---

## 🛠️ Tech Stack

* ☕ Java
* 🔄 Loops & Conditions
* 📦 Arrays
* 🎲 Random Class

---

## 🎯 Game Rules

* 🪨 Rock beats Scissors
* 📄 Paper beats Rock
* ✂️ Scissors beats Paper

---

## ▶️ How to Run

```bash
git clone https://github.com/your-username/Rock-Paper-Scissors.git
cd Rock-Paper-Scissors
javac RockPaperScissors.java
java RockPaperScissors
```
